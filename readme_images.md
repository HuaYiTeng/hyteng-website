## 🎨 纯小白图片处理与压缩金法则

1. **统一1:1正方形原则**：
   除了首屏 Banner 和车间横图，所有产品卡片切图（`desktop-adapter-ip20.png`, `wall-plug-adapter.png`, `ip44-christmas-power-supply.png` 等）在 Publii 的网格里展示时，**强烈建议全部处理成 1:1 (如 600x600 像素) 的正方形透明 PNG**。
   * *做法*：在 Photopea 中创建一个 `600x600px` 的透明画布，然后把导出的产品免抠切图拖进去，完美居中，边缘留出大约 10% 的空白。

2. **背景透明无损 PNG**：
   所有适配器产品图必须是透明背景（Alpha通道），这样在 Publii 网站的主色调（如微灰背景 `#F4F6F8`）下才不会出现难看的白色方形大框。

3. **TinyPNG 终极瘦身**：
   从 Photopea 导出的图体积往往比较大，极易拖慢独立站加载速度。
   * *极简步骤*：打开 **[TinyPNG 官网](https://tinypng.com/)**，把本文件夹下的所有图片批量拖进去。
   * *效果*：图片质量 100% 毫无肉眼可见损失，但文件大小平均减少 **70%**！压缩后，直接覆盖替换本目录下的同名文件，然后再上传到 Publii 中。
